Be neutral when editing Wikipedia
\n

Wikipedia strives to be neutral (NPOV). Remember, you have a point of view (POV). Think about whether your edits will breach this. If you think other users will think it is POV (Wikijargon for "biased"), then run it past other editors using the talk page for the article. This will allow for constructive debate and editing rather than an edit war.\n
