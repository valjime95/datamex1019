Citation styles for "Data science"

\n

APA style

\n

Wikipedia contributors. (2019, October 27). Data science. In Wikipedia, The Free Encyclopedia. Retrieved 00:00, November 8, 2019, from https://en.wikipedia.org/w/index.php?title=Data_science&oldid=923200162\n

\n

MLA Style Manual

\n

Wikipedia contributors. "Data science." Wikipedia, The Free Encyclopedia. Wikipedia, The Free Encyclopedia, 27 Oct. 2019. Web. 8 Nov. 2019.\n

\n

MHRA style

\n

Wikipedia contributors, \'Data science\', Wikipedia, The Free Encyclopedia, 27 October 2019, 01:03 UTC, <https://en.wikipedia.org/w/index.php?title=Data_science&oldid=923200162> [accessed 8 November 2019]\n

\n

Chicago style

\n

Wikipedia contributors, "Data science," Wikipedia, The Free Encyclopedia, https://en.wikipedia.org/w/index.php?title=Data_science&oldid=923200162 (accessed November 8, 2019).\n

\n

CBE/CSE style

\n

Wikipedia contributors. Data science [Internet]. Wikipedia, The Free Encyclopedia; 2019 Oct 27, 01:03 UTC [cited 2019 Nov 8]. Available from: \nhttps://en.wikipedia.org/w/index.php?title=Data_science&oldid=923200162.\n

\n

Bluebook style

\n

Data science, https://en.wikipedia.org/w/index.php?title=Data_science&oldid=923200162 (last visited Nov. 8, 2019).\n

\n

AMA style

\n

Wikipedia contributors. Data science. Wikipedia, The Free Encyclopedia. October 27, 2019, 01:03 UTC. Available at: https://en.wikipedia.org/w/index.php?title=Data_science&oldid=923200162. Accessed November 8, 2019.\n

\n

BibTeX entry

\n
@misc{ wiki:xxx,\n    author = "{Wikipedia contributors}",\n    title = "Data science --- {Wikipedia}{,} The Free Encyclopedia",\n    year = "2019",\n    url = "https://en.wikipedia.org/w/index.php?title=Data_science&oldid=923200162",\n    note = "[Online; accessed 8-November-2019]"\n  }
\n

When using the LaTeX package url (\\usepackage{url} somewhere in the preamble), which tends to give much more nicely formatted web addresses, the following may be preferred:\n

\n
@misc{ wiki:xxx,\n    author = "{Wikipedia contributors}",\n    title = "Data science --- {Wikipedia}{,} The Free Encyclopedia",\n    year = "2019",\n    howpublished = "\\url{https://en.wikipedia.org/w/index.php?title=Data_science&oldid=923200162}",\n    note = "[Online; accessed 8-November-2019]"\n  }
